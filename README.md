## How to Run Locally?

Clone the project

```bash
  git clone https://github.com/dubey2709/FlutterWeb.git
```

Go to your current project directory

```bash
  cd my-project
```

Install dependencies and run the command in the terminal

```bash
   flutter pub get
```

Start the application as a website

```bash
  flutter run -d chrome
```
